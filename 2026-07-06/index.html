AI Builders Digest — 2026-07-06

🔥 新产品发现​

openai/codex-plugin-cc (今日 1519⭐) — OpenAI 官方发布的 Codex ↔ Claude Code 双向桥接插件，让 Codex 能在 Claude Code 中审阅代码或委派任务。标志着编码 agent 之间互操作的重要一步。

usestrix/strix (今日 1121⭐) — 开源 AI 渗透测试工具，自动发现和修复应用安全漏洞。支持多种扫描模式和报告生成。

JuliusBrussee/caveman (今日 1043⭐) — 一个 Claude Code skill，通过让 agent 用"原始人语"减少 65% tokens。"why use many token when few token do trick"——简单粗暴但有效。

Leonxlnx/taste-skill (今日 850⭐) — 给 AI 注入"品味"的 skill，阻止 agent 生成无聊、模板化的内容。在 AI 输出同质化严重的当下，这种追求 quality signal 的工具很有价值。

CoplayDev/unity-mcp (今日 415⭐) — Unity 游戏引擎的 MCP 桥接，让 AI 助手直接管理资产、控制场景、编辑脚本和自动化 Unity Editor 中的任务。

TauricResearch/TradingAgents (今日 257⭐) — 基于多 agent LLM 的金融交易框架，支持策略编排、风险管理等模块。

steipete/CodexBar (今日 201⭐) — Show usage stats for OpenAI Codex and Claude Code, without having to login. macOS menu bar 小工具，展示 token 用量和费率限制。

JulianQuinn/TaskPeace (HN 7👍 7💬) — 通过 MCP 协议让 AI coding agent 从任务队列中 pull 工作，实现多 agent 协作的工作分配和状态管理。

📰 博客更新​

Claude Blog — Building intelligent apps for Apple platforms with Claude in the Foundation Models framework

Anthropic 发布了一个新的 Swift package，允许 Apple 开发者通过 Apple 的 Foundation Models 框架原生调用 Claude。开发者可以在三行代码内调用 Claude 进行多步推理和代码生成，同时保持 Apple 本地模型的高效处理。支持 iOS 27、iPadOS 27、macOS 27、visionOS 27 和 watch OS 27。这意味着 Claude 可以被嵌入到 Apple 原生应用中，从日记应用到学习应用，实现"本地模型处理轻量任务 + Claude 处理复杂推理"的混合架构。

🐦 开发者动态​

Thibault Sottiaux (OpenAI Codex) 在 X 上发问："Codex 至今仍然做不好的事情是什么？" 获得 1804👍 和 1969 条回复，社区反馈集中在复杂多步编排、企业级权限控制和长尾边缘场景。另有一条 "iykyk" 暗示，获 893👍，可能指向即将发布的更新。

Cat Wu (Anthropic Claude Code + Cowork) 分享了 Claude Fable 5 的一个惊喜表现：在用户未指示的情况下，模型自动使用了 propensity score matching（倾向评分匹配）进行用户留存分析，按照活动水平匹配用户组以实现同类比较。她评价："Fable 5 在从 Cowork 邮件写作到 Claude Code 复杂调试的各个环节都展现出了更好的判断力。" 484👍

Guillermo Rauch (Vercel CEO) 发布了一段动态可视化，展示 Vercel AI Gateway 累积的 token 消耗竞赛——该网关聚合了数百万开发者的数万亿 tokens。数据显示 Anthropic 占据主导地位，而开源权重模型正在快速崛起。769👍

Nan Yu (Linear product head) 调侃道："以前手写代码时，心流状态下我嘴里不断吐脏话。所以现在 Codex 做的其实就是 AGI。" 还分享了找 bug 的心得：真正有效的办法是亲自用产品、尝试去破坏它，而不是对代码做"dead reckoning"。

Peter Steinberger (OpenClaw) 预告 CodexBar 即将更新——显示 rate limit 重置时间，帮助用户优化 valuemaxxing 策略。1434👍

🎙️ 播客摘要​

The MAD Podcast with Matt Turck — Cloudflare CEO: The Internet's Business Model Is Dead

https://www.youtube.com/watch?v=UN47z_opfmo

The Takeaway: 2026 年上半年，互联网上的 bot/AI agent 流量首次超过人类流量。Cloudflare CEO Matthew Prince 认为未来五年互联网商业模式将发生根本性变革，因为"bots don't click on ads"。

Matthew Prince 分享了 Cloudflare 从 AI 基础设施公司视角观察到的趋势：作为处理全球大量互联网流量的基础设施公司，Cloudflare 看到了 bot 流量的爆炸式增长——"五年后互联网流量可能是今天的 1000 倍"。在此背景下，大多数公司将在未来 6-12 个月内经历类似 Cloudflare 的团队瘦身（rightsizing）过程，以适应新的 AI 时代。Prince 还深入讨论了 Cloudflare Workers、AI Gateway 和 agent 安全如何成为公司新的增长支柱。他讲述的从初创企业到上市公司的历程中不乏夸张轶事，但核心论点清晰有力：广告支撑的互联网模式正在被 AI agent 流量终结，而企业需要为"后广告"时代的互联网做好准备。
