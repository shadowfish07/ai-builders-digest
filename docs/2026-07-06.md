# AI Builders Digest — 2026-07-06

## 🔥 新产品发现

**[openai/codex-plugin-cc](https://github.com/openai/codex-plugin-cc)** (今日 1519⭐) — OpenAI 官方发布的 Codex ↔ Claude Code 双向桥接插件，让 Codex 能在 Claude Code 中审阅代码或委派任务。标志着编码 agent 之间互操作的重要一步。

**[usestrix/strix](https://github.com/usestrix/strix)** (今日 1121⭐) — 开源 AI 渗透测试工具，自动发现和修复应用安全漏洞。支持多种扫描模式和报告生成。

**[JuliusBrussee/caveman](https://github.com/JuliusBrussee/caveman)** (今日 1043⭐) — 一个 Claude Code skill，通过让 agent 用"原始人语"减少 65% tokens。"why use many token when few token do trick"——简单粗暴但有效。

**[Leonxlnx/taste-skill](https://github.com/Leonxlnx/taste-skill)** (今日 850⭐) — 给 AI 注入"品味"的 skill，阻止 agent 生成无聊、模板化的内容。在 AI 输出同质化严重的当下，这种追求 quality signal 的工具很有价值。

**[CoplayDev/unity-mcp](https://github.com/CoplayDev/unity-mcp)** (今日 415⭐) — Unity 游戏引擎的 MCP 桥接，让 AI 助手直接管理资产、控制场景、编辑脚本和自动化 Unity Editor 中的任务。

**[TauricResearch/TradingAgents](https://github.com/TauricResearch/TradingAgents)** (今日 257⭐) — 基于多 agent LLM 的金融交易框架，支持策略编排、风险管理等模块。

**[steipete/CodexBar](https://github.com/steipete/CodexBar)** (今日 201⭐) — Show usage stats for OpenAI Codex and Claude Code, without having to login. macOS menu bar 小工具，展示 token 用量和费率限制。

**[JulianQuinn/TaskPeace](https://taskpeace.com/)** (HN 7👍 7💬) — 通过 MCP 协议让 AI coding agent 从任务队列中 pull 工作，实现多 agent 协作的工作分配和状态管理。

## 📰 博客更新

**Claude Blog** — **[Building intelligent apps for Apple platforms with Claude in the Foundation Models framework](https://claude.com/blog/claude-for-foundation-models)**

Anthropic 发布了一个新的 Swift package，允许 Apple 开发者通过 Apple 的 Foundation Models 框架原生调用 Claude。开发者可以在三行代码内调用 Claude 进行多步推理和代码生成，同时保持 Apple 本地模型的高效处理。支持 iOS 27、iPadOS 27、macOS 27、visionOS 27 和 watch OS 27。这意味着 Claude 可以被嵌入到 Apple 原生应用中，从日记应用到学习应用，实现"本地模型处理轻量任务 + Claude 处理复杂推理"的混合架构。

## 🎙️ 播客摘要

**The MAD Podcast with Matt Turck — Cloudflare CEO: The Internet's Business Model Is Dead**

[https://www.youtube.com/watch?v=UN47z_opfmo](https://www.youtube.com/watch?v=UN47z_opfmo)

**The Takeaway**: 2026 年上半年，互联网上的 bot/AI agent 流量首次超过人类流量。Cloudflare CEO Matthew Prince 认为未来五年互联网商业模式将发生根本性变革，因为"bots don't click on ads"。

Matthew Prince 分享了 Cloudflare 从 AI 基础设施公司视角观察到的趋势：作为处理全球大量互联网流量的基础设施公司，Cloudflare 看到了 bot 流量的爆炸式增长——"五年后互联网流量可能是今天的 1000 倍"。在此背景下，大多数公司将在未来 6-12 个月内经历类似 Cloudflare 的团队瘦身（rightsizing）过程，以适应新的 AI 时代。Prince 还深入讨论了 Cloudflare Workers、AI Gateway 和 agent 安全如何成为公司新的增长支柱。他讲述的从初创企业到上市公司的历程中不乏夸张轶事，但核心论点清晰有力：广告支撑的互联网模式正在被 AI agent 流量终结，而企业需要为"后广告"时代的互联网做好准备。
