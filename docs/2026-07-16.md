## 🎙️ 播客摘要

### [Training Data：Anthropic's Katelyn Lesse & Angela Jiang: Building an Ecosystem, not a Walled Garden](https://www.youtube.com/watch?v=vPnVTHYplrQ)

**一句话 takeaway：** Anthropic 希望把 platform 从单纯的 Messages API 推向由 knowledge、execution、coordination 组成的可组合 agent 基础设施，而不是把用户锁在 walled garden 里。

Katelyn Lesse 与 Angela Jiang 解释了 Anthropic Platform 的核心取向：内部要给产品团队足够的 speed 和 leverage，外部则要让任何 builder 都能用 Claude 构建自己的产品。平台提供从 messages API、tools、skills、memory 到 Managed Agents 的分层 primitives，同时通过 self-hosted sandboxes、MCP tunnels 以及对 Modal、Vercel、Cloudflare 等基础设施的连接，保持架构的 modularity。她们强调，未来 agent 的关键不只是把模型变大，而是把 token 分配给不同工作：一个 token 执行，另一个负责 advising、反思、grading 或写入 memory。

访谈中的重要判断是，harness 正从“把模型从 A 引导到 B”的脚手架，转向支持更长时间运行和更高层协调的 strategy。工程实践仍然包括 prompt caching、清理 context window 和 evals，但更大的收益可能来自 meta-harness：根据任务复杂度分配 model、token budget 和验证流程。她们还强调开放生态的价值，认为 skills、MCP 与 agent 间的互操作标准，最终会让不同公司的 agent 能像组件一样协作。
