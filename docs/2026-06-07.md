# AI Builders Digest — 2026-06-07

## 新产品

本周最值得关注的 AI 新产品和开源项目：

- **[Lowfat](https://github.com/zdk/lowfat)** — 可插拔的 CLI 过滤器，号称能为 LLM 调用节省 91.8% 的 token 消耗。通过智能过滤无用输入，显著降低 API 成本，对重度使用 AI 的开发者非常实用。**[HN 148👍]**

- **[Boxes.dev](https://boxes.dev)** — 抛弃 localhost，直接在云端运行 Claude Code 和 Codex。为 AI 编码 agent 提供隔离、可扩展的云端运行环境，解决本地资源瓶颈问题。**[HN 103👍]**

- **[CopilotKit](https://github.com/CopilotKit/CopilotKit)** — Agent 与 Generative UI 的前端技术栈。支持 React、Angular、Mobile、Slack 等多平台，内置 AG-UI 协议，让开发者轻松构建具有 AI 交互能力的应用。**[GitHub Trending]**

- **[MemPalace](https://github.com/MemPalace/mempalace)** — 自称是基准测试最佳的开源 AI 记忆系统，完全免费。为大语言模型提供持久化记忆能力，可用于构建具有长期上下文的 AI 应用。**[GitHub Trending]**

- **[Cost.dev (YC W21)](https://cost.dev/)** — 让 AI agent 具备成本意识，自动选择更便宜的调用策略。通过监控和优化 token 使用，帮助开发者在保证质量的前提下减少 API 支出。**[HN 36👍]**

- **[Jo](https://github.com/typescope/jo)** — AI 原生的编程语言，设计目标是在编译期捕获 prompt injection 攻击。从语言层面解决 AI 安全的核心痛点，思路独特。**[HN 10👍]**

- **[Chatcode](https://chatcode.dev/)** — Claude Code 和 Codex 的远程控制工具。允许开发者通过 Web 界面远程管理 AI 编码 agent 的会话，提升多 agent 工作流的可操作性。**[HN 9👍]**

- **[Hydron](https://www.hydron.sh/)** — 硬件感知的编码 agent，能根据当前硬件的 GPU/CPU 情况自动优化编码任务的执行策略，面向异构计算场景。**[HN 9👍]**
