# AI Builders Digest — 2026-06-29

## 🧠 模型发布

**[GPT-5.6 Sol / Terra / Luna](https://openai.com/index/previewing-gpt-5-6-sol/)** — OpenAI 于 6 月 26 日发布 GPT-5.6 系列，首次将旗舰模型拆分为三个命名层级：Sol（旗舰推理 + 长周期 agent 工作，$5 输入 / $30 输出每百万 token）、Terra（均衡日常模型，性能对标 GPT-5.5 但价格便宜 2 倍，$2.50/$15）、Luna（最快最便宜，$1/$6）。Sol 还提供高算力模式 Sol Ultra。**可用性**: Limited Preview，仅限受信合作伙伴通过 API 和 Codex 访问，美国政府的限制发布要求是此次非全面开放的主要原因。7 月还将在 Cerebras 上以 750 tok/s 的速度运行 Sol。**评价**: GPT-5.5 仅两个月后的又一次迭代，"多模型家族"的定价策略标志着 OpenAI 从单一大模型走向分层产品矩阵。

**[Grok 4.5](https://jang.com.pk/en/68261-elon-musk-makes-major-grok-45-announcement-on-55th-birthday-news)** — Elon Musk 在 55 岁生日当天（6 月 28 日）宣布 Grok 4.5 进入 SpaceX 和 Tesla 内部 Beta 测试。基于 1.5 万亿参数的 V9 基础模型，训练中融入了 Cursor 数据。初步评估显示性能已接近或超越 Opus。**可用性**: 内部测试阶段。SpaceX 计划每月从头训练并发布全新模型。**评价**: xAI 在 Grok 4 之后快速迭代，强调 RL 持续优化和月级发布节奏，追赶压力直接给到 Anthropic。

**[Ornith-1.0](https://explainx.ai/blog/ornith-1-0-self-scaffolding-agentic-coding-llm-2026)** — DeepReinforce.AI 于 6 月 25 日发布，MIT 开源，覆盖 9B Dense / 31B Dense / 35B MoE / 397B MoE 四个规模。核心创新是 self-scaffolding 训练框架——模型在 RL 中同时学习生成解法和指导解法的 scaffold，而非依赖人工设计的 harness。旗舰 397B 成绩亮眼：Terminal-Bench 2.1 77.5、SWE-Bench Verified 82.4，超越 Claude Opus 4.7 和 DeepSeek-V4-Pro。基于 Gemma 4 和 Qwen 3.5。**可用性**: HuggingFace 开源，MIT 协议。**评价**: 开源编码模型的有力竞争者，"scaffold 是可以训练的"这一理念对 agent 生态意义深远。

## 🎙️ 播客摘要

**Training Data — Memory and Continual Learning: Engram's Dan Biderman and Jessy Lin**

The Takeaway: 当前模型的智能瓶颈不是原始能力，而是理解和融入持续变化的新上下文——真正的记忆不应该是一个单独的外部数据库或塞进 context window 的东西，而是深嵌入模型权重中的持续学习能力。

Engram 联合创始人 Dan Biderman 和 Jessy Lin 挑战了当前 AI 主流范式——预训练和后训练的静态分割。他们认为，让模型真正有用需要突破"一次性训练"的局限："模型今天已经知道很多，但瓶颈在于理解新的、不断演变的上下文。" 无论是工作中的特定语境还是新的任务类型，都需要像预训练一样将知识深嵌入模型权重。

这对所谓的"RAG"和"memory"方案提出了根本性质疑：如果说 context window 里的检索结果不算真正的记忆，那什么才算？Engram 的回答是——持续学习（continual learning），即模型始终处于训练状态，随时吸收新信息并固化到权重中。这意味着未来的 AI 系统将不再是"冻结模型 + API 调用"的静态结构，而是能像人类一样持续演化的动态认知体。
