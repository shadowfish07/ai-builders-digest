## 🎙️ 播客精选

**No Priors — Building an AI Guardian for Enterprise with Onyx Security CEO Maxim Bar Kogan**

本期 No Priors 的"Takeaway"：随着 AI agent 在企业中的采用呈指数级增长，agent 安全已成为市场的核心焦虑——Onyx Security 正是抓住了从 AutoGPT 到 Claude Code 的 agent 演进浪潮，训练专门的模型来监控其他 AI agent 的行为。

Maxim Bar Kogan 回忆，公司的灵感来自于 AutoGPT——第一个真正自主运行在 LLM 上的 agent，虽然当时模型还不够成熟，但让他预见到了未来：当模型足够聪明时，需要有人来监督这些"比我们更聪明"的 agent。如今，Claude Code 已成为企业广泛采用的首个自主编码 agent，Anthropic 的大部分企业收入来自 Claude Code。

Onyx 的产品做两件事：1）训练模型和构建 agent 来监督其他 agent 的行为；2）将这些能力产品化为"AI 控制平面"(Secure AI Control Plane)，连接到企业的所有 AI 和 agent，监控它们的行动是否合法。因为 agent 正在以指数级增长，人工审核（Human-in-the-loop）已经无法跟上——当你有 100x、1000x、甚至百万倍的 agent 动作时，必须用 AI 来监督 AI。

Maxim 还讨论了 agent 可能带来的真实风险：误发布代码和 token、意外删除数据、造成服务宕机等。企业意识到风险正在急剧上升，但又无法阻止 agent 的采用，只能寻找降低风险的方法。
