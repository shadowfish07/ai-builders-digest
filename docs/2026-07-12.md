## 🎙️ 播客摘要

### Stripe's AI Chief: How AI Agents Will Buy, Sell, and Pay

**The Takeaway**：agentic commerce 的关键变化不是让个人授权 agent 买东西，而是让 agent 作为业务主体持续执行采购、销售和盈利，软件的计费与信任模型也必须随之改变。

Stripe 数据与 AI 负责人 Emily Sense 认为，未来的 agent 不只是替人下单，而可能负责经营一整个业务：它会代表公司购买资源、销售服务、管理预算，并根据结果持续优化。传统的 per-seat pricing 因此正在失效，因为真正消耗价值的单位从“人”转向了 agent 完成的工作量与 token 使用量。

她特别强调了一个常被忽略的安全问题：AI 公司面对的滥用不一定是凭证或资金被盗，也可能是 token 被盗用。她把这种行为比作“token 版的 dine and dash”，并指出 AI 公司每六个注册中就可能有一个属于这类滥用。对 agent commerce 来说，权限、预算上限、可撤销授权和交易审计不是附加功能，而是经济基础设施的一部分。

最值得注意的判断是，agent 的最终形态可能不是“替 Emily 买东西”，而是“Emily 的 agent 负责运营一门生意”。这会把身份、支付、计费和企业软件全部重新连接起来。
