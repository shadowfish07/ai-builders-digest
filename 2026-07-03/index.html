AI Builders Digest — 2026-07-03

🧠 模型发布​

Claude Fable 5 全球恢复上线 — 美国商务部正式解除对 Claude Fable 5 和 Mythos 5 的出口管制，Anthropic 于 7 月 1 日恢复全球访问。新增安全分类器可将特定越狱手法阻断率提升至 99% 以上，被拦截请求自动降级至 Opus 4.8。付费用户可在 7 月 7 日前使用 Fable 5（最多占用 50% 周用量配额）。评价: 经历三周出口管制风波后，Fable 5 重回市场，标志着美国 AI 出口管制框架进入"先审后放"新阶段。

Kimi K2.7 Code 登陆 GitHub Copilot — Moonshot AI 的开源编码模型成为 Copilot 模型选择器中首个开源权重选项。已在 Copilot Pro/Pro+/Max 计划中逐步推送，支持 VS Code、VS、JetBrains、Xcode 等 IDE。评价: 开源模型首次进入 Copilot 官方候选，开发者编码工具的多模型选择正式开启。

⚙️ 工具更新​

Claude Code v2.1.199 Changelog

  • 支持连续斜杠技能调用（/skill-a /skill-b），最多加载 5 个技能
  • 修复 SSL 证书错误（TLS 代理、NODE_EXTRA_CA_CERTS 缺失、过期证书）导致的重试烧 Token 问题，现在直接报错并提示修复方案
  • 修复 API 中途返回 overloaded/server error 时流式响应被丢弃的问题，保留已输出部分并标记不完整
  • 修复子 agent 因速率限制或服务器错误被切断时静默失败的问题，现在返回部分工作成果
  • 修复子 agent 报告 API 错误（如用量超限）时伪装成成功结果的问题
  • 修复 Linux 上后台 agent 守护进程在非正常关闭后每 ~50 秒自杀的问题
  • 修复 macOS 上 SSH 后台 agent 冷启动失败的问题
  • 修复 claude stop 与后台 agent 重启竞态导致停止被忽略的问题

Claude Code Dynamic Workflows 正式 GA — Anthropic 将 Dynamic Workflows 从研究预览升级为正式版，Pro 用户现在也可使用（默认关闭，需手动启用）。支持最多 1000 个并行子 agent，此前 Bun 运行时从 Zig 到 Rust 的 96 万行移植即基于此能力完成。

AI & I by Every — The AI Workflows Behind Every's Consulting Team

🎧 收听

The Takeaway: AI 正在将知识工作从"用手制造"转变为"像园艺一样创造条件"——大公司高管已经开始用 Codex 构建自己的"操作系统"，让助手比本人更了解自己的工作情况。

Every 的咨询主管 Natalia 回到节目，讲述她如何用 AI 工具（特别是 Codex）为大型企业高管赋能。她的核心洞察是：知识工作正在从"用手制造"转变为"像园艺一样创造条件"——你不再亲手种植植物，而是创造让植物生长的条件。Codex 帮她构建了一个"操作系统"，让 AI 助手比她自己更了解她的工作流。最精彩的一句话来自开场："你教高管和大公司的人如何使用 AI，你做的这件事是了解优秀运营者和高管如何开始使用这些东西的一个很好的窗口。"她提到对所有行政性任务即将被 AI 接管感到极度乐观——因为我们现在有了"超级外星工具"来支持这些事。
