🧠 模型发布​

Mistral 3 全系列开源：675B Large 3 MoE + Ministral 3 边缘模型 + Voxtral TTS​

Mistral AI 在 6 月 18 日发布了 Mistral 3 模型家族，涵盖从 3B 到 675B 的全尺寸范围，全部采用 Apache 2.0 许可证，是迄今为止最大规模的完全开源模型发布。

  • Mistral Large 3：675B 总参数 / 41B 激活参数的稀疏 MoE 架构，使用 3,000 张 NVIDIA H200 GPU 从头训练。支持 256K 上下文窗口、多模态（图像理解）、多语言。在 LMArena 开源非推理模型中排名第 2，在所有开源模型中排名第 6。推理版本即将推出。输入 / 输出定价 $0.50 / $1.50 每百万 token。
  • Ministral 3 系列：3B / 8B / 14B 三个尺寸，每个有 Base、Instruct、Reasoning 变体。14B Reasoning 变体 AIME 2025 得分 85%。
  • Voxtral TTS：4B 参数文本转语音模型，支持 9 种语言，70ms 延迟，3 秒音频即可克隆声音，定价 $0.016/千字符。Hugging Face 上以 CC BY-NC 4.0 发布。
  • 同时发布了 Mistral OCR 3（文档提取，$2/千页）和 Leanstral（Lean 4 证明助手，6B 激活 / 120B 总参数，Apache 2.0）。
  • 可用性：已在 Mistral AI Studio、Amazon Bedrock、Azure Foundry、Hugging Face、Modal、Together AI 等平台上线，NVIDIA NIM 即将支持。
  • 评价：Apache 2.0 下的 675B 开源模型，性能与效率兼备，是开源社区的重大里程碑。

📰 博客更新​

Anthropic Engineering（10 篇精选）​

Anthropic 工程博客本周密集发布了关于 agent 安全、系统设计和评估实践的技术文章：

  • How We Contain Claude Across Products — 详细阐述了为 claude.ai、Claude Code 和 Claude Cowork 构建安全隔离的工程实践：通过沙箱、虚拟机、出口控制来限制 agent 的爆炸半径，而非仅依赖人类监督。指出用户 93% 的权限请求会被批准，导致批准疲劳，因此需要更智能的自动防御机制。
  • How We Built Claude Code Auto Mode — 介绍了 auto mode 的双层防御架构：输入层 prompt 注入检测 + 输出层基于 Sonnet 4.6 的转录分类器，在跳过权限审批的同时拦截危险操作。
  • Scaling Managed Agents — 将 agent 的大脑（Claude + harness）与手（sandbox + tools）和会话日志解耦，实现容器变成 cattle 而非 pet 的架构转变。
  • 另有 April 23 Postmortem、Harness Design for Long Running Apps、Eval Awareness BrowseComp、Infrastructure Noise、Building C Compiler、AI-Resistant Technical Evaluations、Demystifying Evals for AI Agents 等。

🎙️ 播客摘要​

No Priors: Re-engineering the Semiconductor Supply Chain with Intel CEO Lip Bu Tan​

YouTube 链接

核心观点：66 岁的半导体传奇人物 Lip Bu Tan 为何接管 Intel 这个行业最难的工作？答案不是财务回报，而是使命感。

Intel CEO Lip Bu Tan（前 Cadence CEO、Walden 创始人）在 No Priors 播客中分享了其 Intel 复兴计划。他采取三步走策略：强化资产负债表、聚焦产品、倾听客户后推动下一代领导性产品。他强调了「匍匐前进、保持谦逊」的管理哲学——他投资的 10 家公司中有 9 家中途改变了商业计划，因此他更看重具有韧性的创业者团队。

关于半导体供应链，他透露 agentic AI 和 inference CPU 的需求正在激增，这恰好是 Intel 的机遇。老友 Jensen Huang（NVIDIA CEO）投资的 50 亿美元目前已增值至 250 亿美元，标志着两大芯片巨头在 AI 时代的深度绑定。

关于美国政府作为主要股东的角色，Tan 保持了务实态度——Intel 对美国的半导体生态系统和国家安全至关重要，这是他选择接手而非退休的根本原因。值得关注的是他提到的「从爬到走再到跑」的节奏，以及如何将 Intel 从一家应对市场的公司转变为定义市场的公司。
