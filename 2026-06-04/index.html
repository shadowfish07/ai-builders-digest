🎙️ 播客精选 (Podcast)​

No Priors — The Rise of the Full-Stack Builder with Microsoft CEO Satya Nadella

Satya Nadella 在 Microsoft Build 2026 后接受 No Priors 专访，深入分享微软的 AI 平台战略。

关键洞察：

  1. MAI 模型的清洁血统比基准分数更重要——很多开源模型在某两个基准上好看但实际不好用
  2. 企业真正的 IP 是私有 eval，能用模型 A hill climb 后无缝切换到模型 B 是控制权的试金石
  3. 编码 AI 太成功以至于需要重新构建 IDE——100 个 agent session 的认知负载让 chat 不再是唯一的交互范式
  4. 每家公司最终会拥有「公司老兵 agent」，其累积的 tacit knowledge 可能首次实现知识资产上资产负债表

Nadella 认为 AI 将创造更多就业——代码效率提升带来更多软件项目，销售和市场因 agent 处理更多线索而扩招。
